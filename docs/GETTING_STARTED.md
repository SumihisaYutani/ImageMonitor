# 🚀 ImageMonitor - はじめに

## 📖 ImageMonitorとは？

**ImageMonitor**は画像ファイルとアーカイブ（ZIP/RAR）内の画像を一括管理できるツールです。

### ✨ 主な機能
- 📁 複数フォルダの画像を一覧表示
- 📦 ZIP/RAR内の画像も表示・検索
- 🔍 高速検索・フィルタリング
- 🖼️ サムネイル表示
- 🎯 アーカイブ内画像比率での絞り込み

---

## ⚡ 5分でスタート

### 手順1: フォルダを設定
1. アプリを起動
2. **ファイル** → **設定** をクリック
3. **「追加」**ボタンで画像があるフォルダを選択
4. **「OK」**で保存

### 手順2: スキャン実行
1. **「Scan」**ボタンをクリック
2. 完了まで待機（進捗は下部に表示）

### 手順3: 画像を楽しむ！
- サムネイルをダブルクリックで画像を開く
- 検索ボックスでファイル名検索
- フィルターで表示内容を調整

---

## 🎛️ 重要なコントロール

### 「Include Archives」チェックボックス
```
✅ ON  → ZIP/RAR内の画像も表示（推奨）
❌ OFF → フォルダ内の画像のみ表示
```

### 「Image Ratio」スライダー
```
高い値（70-100%） → 画像中心のアーカイブのみ
低い値（10-30%）  → 画像が少ないアーカイブも含む
```

---

## 💡 よくある使い方

### パターン1: 全ての画像を見たい
1. **Include Archives**: ✅ ON
2. **Image Ratio**: 50%程度
3. **検索ボックス**: 空白

→ フォルダとアーカイブ両方の画像が表示

### パターン2: フォルダの画像のみ見たい
1. **Include Archives**: ❌ OFF  
2. **Image Ratio**: 関係なし
3. **検索ボックス**: 空白

→ フォルダ内の画像のみ表示

### パターン3: 特定の名前の画像を探したい
1. **Include Archives**: ✅ ON
2. **Image Ratio**: 50%程度  
3. **検索ボックス**: 「キャラ名」や「シリーズ名」を入力

→ 名前に該当する画像が絞り込み表示

---

## ❓ 困った時は

### 画像が表示されない
1. フォルダの設定を確認
2. スキャンが完了しているか確認
3. **Include Archives**をONにしてみる

### アーカイブ内画像が見えない  
1. **Include Archives**をONに
2. **Image Ratio**を下げる（20%程度）
3. ZIP/RAR形式か確認

### 動作が重い
1. **Image Ratio**を上げる（70%以上）
2. サムネイルサイズを小さく
3. 不要フォルダをスキャン対象から除外

---

## 🔧 おすすめ設定

### 初心者向け設定
- **Include Archives**: ✅ ON
- **Image Ratio**: 50%  
- **サムネイルサイズ**: 128px
- **ログレベル**: Information

### 上級者向け設定  
- **Include Archives**: 使い分け
- **Image Ratio**: 用途に応じて調整
- **サムネイルサイズ**: 192px以上
- **ログレベル**: Warning以上
